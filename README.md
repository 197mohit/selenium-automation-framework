# 🚀 Selenium Automation Framework

A production-ready Selenium automation framework developed using Java and TestNG.

## Features

- Selenium 4
- Java
- TestNG
- Maven
- Page Object Model (POM)
- Explicit Waits
- Retry Mechanism
- Extent Reports
- Log4j
- Parallel Execution
- GitHub Actions
- Jenkins Integration
